# Cisco Packet Tracer Labs

This repository contains my Cisco Packet Tracer practice activities completed while learning Computer Networks and Cisco Networking Academy.

## 📚 Modules Completed

### Module 1: Create a Simple Network

### Objective
Build a basic home network using Cisco Packet Tracer and verify connectivity between devices.

### Devices Used
- Wireless Router
- PC
- Laptop
- Cable Modem
- Internet Cloud
- Cisco Server

### Tasks Performed
- Added network devices to the workspace
- Connected devices using appropriate cables
- Configured end devices
- Verified connectivity
- Explored logical network topology

### Skills Learned
- Network topology
- Ethernet connections
- Wireless networking
- Basic IP connectivity
- Packet Tracer interface

---

### Module 2: Logical and Physical Mode Exploration

### Objective
Understand the difference between Logical Mode and Physical Mode in Cisco Packet Tracer.

### Tasks Performed
- Explored Logical Workspace
- Explored Physical Workspace
- Navigated different physical locations
- Observed racks, wiring closets, tables, and shelves
- Compared logical and physical network representations

### Skills Learned
- Logical topology
- Physical topology
- Device placement
- Network infrastructure visualization
